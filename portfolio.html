ReadMe Generator

This application helps you to save time with your README documentation leaving you with more time to focus on your project.

It uses Node.js to run prompts in the command line using the user interface provided by Inquirer.js. A professional README is built using the user response, through Node.js standard library package fs a new md file is created. Same principle can be used to build and write different types of files as an entire html web application.
